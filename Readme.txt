# Análise de Cancelamento de Clientes

Este projeto de análise de dados em Python tem como foco entender os principais motivos que levam os clientes de uma empresa a cancelarem seus serviços. Através da análise dos dados de cancelamentos, busca-se desenvolver estratégias eficazes para reduzir a taxa de cancelamento e melhorar a retenção de clientes.

## Dados

O conjunto de dados `cancelamentos_sample.csv` contém informações de 50.000 clientes, incluindo dados demográficos, detalhes do contrato, uso do serviço, interações com o call center, atrasos nos pagamentos, e a informação crucial de se o cliente cancelou ou não o serviço.

## Pré-requisitos

Para executar este projeto, você precisará de:

- Python 3.x
- Jupyter Notebook ou JupyterLab
- Bibliotecas: pandas, numpy, matplotlib, seaborn

Você pode instalar todas as dependências necessárias usando o seguinte comando:

```bash
pip install jupyterlab pandas numpy matplotlib seaborn
```

## Como Usar

1. Clone ou faça o download deste repositório para o seu computador local.
2. Certifique-se de que todos os pré-requisitos estão instalados.
3. Abra o arquivo `inicial.ipynb` usando o Jupyter Notebook ou JupyterLab.
4. Execute as células em sequência para ver a análise e os insights gerados.

## Estrutura do Código

O notebook `inicial.ipynb` inclui várias seções que guiam você através do processo de análise de dados, desde a carga e limpeza dos dados até a exploração e visualização de insights significativos relacionados aos cancelamentos de clientes.
